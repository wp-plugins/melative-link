The melative-link plugin allows easy mouseover displays of title information from melative.com.

== Description ==

The melative-link plugin allows easy mouseover "Bubble" displays of title information from <a href="http://melative.com">melative.com</a>. Melative holds light meta data for works of art and entertainment.

== Installation ==

1. Upload the entire `melative-link` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

Please see the <a href="http://melative.pbworks.com/WordPress" title="Plugin homepage">plugin homepage</a> for instructions on how to use this plugin.
